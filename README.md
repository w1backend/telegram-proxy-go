# Telegram Proxy Service (Go)

Простой и эффективный прокси-сервер на языке Go. 

### Особенности:
- Написано на Golang (высокая производительность).
- Поддержка Telegram протоколов.
- Легкая настройка и запуск через Docker (в планах).

### Как запустить:
```bash
go run main.go
